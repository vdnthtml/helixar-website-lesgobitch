# Stitch Prompt Templates — Helixar Landing Page

---

## How To Use These

Copy the entire prompt block. Paste it into Google Stitch exactly as written.
Don't summarize. Don't paraphrase. The specificity is what gets good output.

## PROMPT 1 — Hero Section

```
Design a landing page hero section for "Helixar" — an AI marketing workspace for Mac and Windows.

VISUAL STYLE:
- Dark background: #0A0A0B (near black)
- Surface cards: #111113
- Brand accent: #FF5C00 (orange — used sparingly, only on CTAs and logo)
- Typography: Geist font family
- Aesthetic: Linear.app + Raycast energy — precise, dark, quiet confidence

LAYOUT:
- Fixed top navigation: logo left, 3 nav links center, one orange CTA button right
- Hero: centered content above the fold, product screenshot below
- Maximum content width: 1200px, centered

HERO CONTENT (use exactly this copy):
- Small eyebrow badge: "AI MARKETING WORKSPACE" in uppercase, subtle orange tint background
- Headline (large, 64–80px): "Marketing used to need a team. Now it needs you."
- Subhead (18px, muted): "Helixar researches competitors, plans campaigns, generates content, and tracks performance — all in one workspace that gets smarter every time you use it."
- Two CTAs: Primary orange button "Download for Mac", secondary text link "See how it works →"
- Below CTAs: App screenshot mockup in a dark card container with subtle border

DETAILS:
- Background has a very subtle dot grid pattern (#ffffff at 3% opacity, 48px grid)
- Orange radial glow in top-left background (very subtle, 8% opacity)
- Navigation bar: backdrop blur, semi-transparent dark background when scrolled
- No gradients except on the logo mark (orange to lighter orange)
- Generous whitespace — this is not cramped
- The product screenshot should show a dark desktop app UI with orange accents

DO NOT:
- Use purple, blue, or teal anywhere
- Use rounded corners larger than 16px on the screenshot container
- Add stock photography or people photos
- Make the design feel "startup generic" — it should feel like a serious professional tool
```

## General Tips When Using Stitch

1. **Always specify exact hex colors.** "Orange" means nothing. "#FF5C00" does.
2. **Name the exact font.** "Geist" not "a modern font."
3. **Give exact copy.** Don't let Stitch invent headline copy — paste yours.
4. **List what NOT to do.** This is as important as what to do.
5. **Reference real products.** "Linear.app aesthetic" gives Stitch a mental model.
6. **Iterate on one section at a time.** Don't ask for the whole page in one prompt.
7. **After first output:** Say "Keep the dark base and orange accents exactly the same, but change [specific thing]."
